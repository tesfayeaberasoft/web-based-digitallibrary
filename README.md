# Digital Library Management System

A comprehensive web-based digital library management system built with React, PHP, and MySQL.

## Features

### Core Features
- User Management (Student, Staff, Admin roles)
- Book Management (Add, Update, Delete, View)
- Search & Browse functionality
- Issue & Return Books workflow
- Real-time availability tracking

### Advanced Features
- Reservation System with queue management
- Fine Management and payment integration
- Notifications system
- Role-Based Access Control (RBAC)
- Reports & Analytics
- Digital Content Support (PDF, EPUB)
- Recommendation System
- Multi-Library Support
- Barcode/QR Code Integration
- Audit Logs
- Payment Integration (Chapa Gateway)

## Technology Stack

- **Frontend**: React.js with modern UI components
- **Backend**: PHP with RESTful API
- **Database**: MySQL
- **Payment**: Chapa Payment Gateway
- **External APIs**: Google Books API, SendGrid, Google OAuth 2.0

## Installation

1. Clone the repository
2. Set up the database using the provided SQL files
3. Configure the backend API endpoints
4. Install frontend dependencies
5. Configure environment variables

## Project Structure

```
├── frontend/          # React frontend application
├── backend/           # PHP backend API
├── database/          # MySQL database schema and data
├── docs/             # Documentation
└── assets/           # Static assets and images
```

## Requirements Implemented

Based on the Digital Library Management System Requirements Specification V4, this system implements all core, intermediate, and advanced features including:

- Complete user management system
- Comprehensive book management
- Advanced search and filtering
- Reservation and fine management
- Payment integration
- Analytics and reporting
- Mobile-responsive design
- Security features
